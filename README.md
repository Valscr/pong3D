# PONG3D

![Capture d'écran du jeu](capture/screen.png)

Game developed using Babylon.js, inspired by the famous PONG game.

Install dependencies:
```bash
npm install
```
Run the game:
```bash
npm run dev
```
Open your browser and navigate to the specified URL to play the game.
